# Personal Expense Tracker - Information Gathering Report (Basic)

## 1. Introduction
This report explains how I gathered information to decide what features this app needs. I wanted to keep it simple so that students and individuals can use it without any confusion.

## 2. Methodology
*   **Self-Observation:** I tracked my own daily expenses for one week to see which categories (like Food, Transport, and Bills) are used most often.

## 3. Findings
*   **Main Categories:** Most users only care about 5-6 simple categories like Food, Rent, Bills, and Fun.
*   **Fast Logging:** Users want to enter an expense quickly (in 2-3 clicks).
*   **Clear Balance:** Everyone wants to see their "Total Balance" and "Recent Transactions" on the home page.

## 4. Features to Build
*   **Login Page:** So each user has their own records.
*   **Entry Form:** To input Amount, Category, and Date.
*   **Visual Chart:** A simple pie chart to show where the money goes.
*   **History List:** A list of all past transactions.
