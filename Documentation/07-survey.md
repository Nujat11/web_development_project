# Personal Expense Tracker - Survey Report (Basic)

## 1. Survey Overview
| Attribute | Value |
|---|---|
| Sample Size | 10 respondents |
| Segments | Students |
| Duration | 2 days |
| Method | Online Form |

## 2. Survey Questions and Results
| Q# | Question | Top Response |
|---|---|---|
| Q1 | Do you currently track your expenses? | No, not at all |
| Q2 | Why do you stop tracking? | Too much effort / Forgetful |
| Q3 | What feature would help you most? | Simple "Add" button |
| Q4 | Do you want to see your total balance? | Yes, very important |
| Q5 | Would you use categories like Food/Rent? | Yes |
| Q6 | Is a Pie Chart helpful to you? | Yes |
| Q7 | How many clicks should it take to log? | 1-2 clicks |
| Q8 | Should the app remember monthly bills? | Yes |

## 3. Feature Demand Ranking
| Rank | Feature |
|---|---|
| 1 | Simple "Add" Transaction |
| 2 | Total Balance Display |
| 3 | Category-wise Pie Chart |
| 4 | Monthly History List |
| 5 | Recurring Bill Reminders |

## 4. Analysis
1. **Simplicity is Key:** People want an app that doesn't feel like work.
2. **Speed Matters:** Logging should be done in a few seconds.
3. **Visuals Help:** A clean Pie Chart is much better than a long list.
4. **Visibility:** The front page must show the current balance clearly.

## 5. Recommendations
| Recommendation |
|---|
| Keep the Home Page focused on the 'Add' button |
| Use large, clear icons for categories |
| Show the 'Total Balance' at the top of the screen |
| Include a simple Pie Chart on the main page |
