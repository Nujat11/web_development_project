# 11 - User Stories 

This table outlines the core functional requirements from the perspective of our users.

| ID | Action (I want to...) | Reason (So that...) | Priority |
|---|---|---|---|
| **US-01** | Create a private account | My financial data remains secure and private. | High |
| **US-02** | Securely log into my account | I can access my data from any device safely. | High |
| **US-03** | Quickly add an expense | I don't forget small spends like rickshaws or snacks. | High |
| **US-04** | Select a category for each spend | I can see a clear breakdown of where my money goes. | High |
| **US-05** | View my 'Total Balance' on home | I know exactly how much money I have left for the day. | High |
| **US-06** | See a visual Pie Chart | I can easily spot which category is eating most of my budget. | Medium |
| **US-07** | View a full history of spends | I can review my spending habits over the past month. | Medium |
| **US-08** | Edit or Delete a transaction | I can correct mistakes if I enter the wrong amount. | Low |
| **US-09** | Reset my password | I can regain access if I ever forget my credentials. | Low |
| **US-10** | Set a daily spending limit | I don't run out of allowance before the month ends. | Medium |
