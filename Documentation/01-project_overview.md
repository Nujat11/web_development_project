# Personal Expense Tracker - Project Overview (Basic)

## 1. Introduction
The Personal Expense Tracker is a simple web application designed to help people track their daily income and expenses. The goal is to provide an easy way for users to see where their money is going and manage their savings effectively.

## 2. Why this project?
Many people struggle to remember their daily small expenditures like snacks, transport, or small bills. Manual tracking on paper is difficult and often ignored. This app provides a digital way to log and visualize these expenses instantly.

## 3. Main Goals
*   Create a private account for each user to store their data.
*   Allow users to easily add, update, or delete transactions.
*   Display a clear "Total Balance" summary.
*   Show a simple visual chart to analyze spending by category (e.g., Food, Travel, Rent).

## 4. Technology Stack
*   **Frontend:** React (for a fast and responsive user interface).
*   **Backend:** FastAPI (to handle calculations and data storage).
*   **Database:** PostgreSQL (to store transaction history safely).
*   **Security:** JWT (to keep user accounts and data private).

## 5. Project Features (Scope)
| **Feature** | **Description** |
| :--- | :--- |
| **User Login** | Secure registration and login for individual users. |
| **Add/Edit Transaction**| Users can log their income or expenses with a few clicks. |
| **Dashboard** | A summary page showing the balance and recent activities. |
| **Spending Charts** | Simple visual graphs to show where the money is being spent. |
| **History List** | A clear list of all previous income and expense entries. |

## 6. Project Outcome
This project will result in a functional web tool that helps individuals improve their financial discipline by making income and expense tracking effortless and visual.
